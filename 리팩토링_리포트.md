## 3. 설정 및 환경

### 설정 파일 목록
1. **`backend/.env`** - R2 스토리지 환경변수
   ```
   R2_ACCESS_KEY_ID=...
   R2_SECRET_ACCESS_KEY=...
   R2_ENDPOINT=...
   R2_BUCKET=obsidian-attachments
   ```

2. **`extension/manifest.json`** - 크롬 확장 설정
   ```json
   {
     "permissions": ["storage"],
     "host_permissions": [
       "http://localhost:11434/*",
       "https://www.genspark.ai/*"
     ]
   }
   ```

3. **`backend/web/wrangler.toml`** - Cloudflare Worker 설정
   ```toml
   name = "aikeep24-web"
   main = "worker.js"
   compatibility_date = "2025-02-25"
   
   [[d1_databases]]
   binding = "DB"
   database_name = "obsidian-db"
   database_id = "..."
   
   [[vectorize]]
   binding = "VECTOR_INDEX"
   index_name = "aikeep24-vectors"
   ```

### 환경변수 사용 현황
- **사용 파일**: `scripts/sync-obsidian-to-d1.py` (R2 접근 정보)
- **패턴**: `os.getenv()`를 통한 환경변수 읽기
- **한계**: Ollama URL, Worker URL 등 주요 설정이 하드코딩됨
